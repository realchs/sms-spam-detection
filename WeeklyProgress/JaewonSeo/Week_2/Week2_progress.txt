//트위터 API

HAM 데이터를 수집

구어체 및 대화형으로 되어 있는 텍스트 데이터를 얻어야 함

검색을 해보니 "근데, 진짜, 했어" 등등 많이 쓰는 구어체 단어들을 키워드로
선정하면 원하는 방향에 근접하는 데이터를 얻을 수 있을 것으로 보여졌음.

지난주에 사용한 라이브러리 이외에 다른 크롤링 라이브러리를 사용해보려고 했으나
잘 작동을 하지 않음
-> 기존 코드에 원하는 구어체 리스트를 만들고 리스트별로 데이터를 얻을 수 있게 약간 수정

뽑혀진 output file을 살펴보니, 예상보다 좀더 자연스러운 대화형 텍스트 데이터를 얻을 수 있었음.
(물론, 잘 쓰지않는 형식이나 광고성 글도 보이긴 해서 어떻게 처리를 할지 고민 필요)
-> 텍스트에 키워드가 들어간 글만 가져오지 않고, 다양한 대화가 저장되어 좋은 것 같음.

output file data 회의때 확인을 받고, 괜찮다면 다양한 키워드로 많이 받아오면 좋을 것 같음.
